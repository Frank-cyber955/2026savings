# 2026年存款计划

个人存款目标追踪与记录工具。

## 功能
- 📊 仪表盘：总览存款进度
- 📦 二手物品管理
- 🛵 外卖收入记录
- 💰 额外收入记录
- 📅 时间线里程碑
- 🏦 对账功能
- 📱 支持添加到手机桌面（PWA）

## 部署到 GitHub Pages

1. Fork 或上传此仓库
2. Settings → Pages → Source: Deploy from a branch
3. Branch: main, 文件夹: / (root)
4. 保存，等待几分钟
5. 访问 `https://你的用户名.github.io/仓库名/`

## 添加到手机

### iPhone
1. Safari 打开网址
2. 点底部分享按钮 ⬆️
3. 选「添加到主屏幕」

### Android
1. Chrome 打开网址
2. 点右上角菜单 ⋮
3. 选「添加到主屏幕」或「安装应用」

## 数据说明
- 数据存储在浏览器 localStorage
- 不同设备间通过导出/导入 JSON 同步
- 清除浏览器数据会丢失记录，建议定期导出备份
